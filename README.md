# b2b-saas-content-research

Research Project: YouTube Content Strategy for B2B SaaS

Why I chose this topic
Video content is becoming one of the major channels B2B SaaS companies use to reach and educate their audience. I wanted to study people who have actually built channels and frameworks in this space, rather than just reading about theory.

How I selected the 10 experts
I avoided generic top 10 listicle names. I cross checked multiple sources, confirmed each channel is currently active with recent uploads, and prioritized people who genuinely create content and practice what they teach rather than agencies or one time guest appearances.

What is inside the research folder
sources.md contains the full list of 10 experts with reasoning for each choice.
youtube-transcripts folder contains summaries of recent videos from each expert.
other folder contains additional notes and observations from the research process.

Tools used
I used Cursor IDE along with Claude Code and Codex to assist with research and documentation. 
I also used manual research on YouTube to find and verify each expert.

Issues I faced and how I solved them
While creating folders in Cursor, empty folders did not get pushed to GitHub since Git does not track empty folders. To solve this, I created the files for folder directly on GitHub, which automatically created the folder structure along with the content.
